# Excel-Escape-Room
Repository for the Excel Escape Room, a gamified learning tool for teaching Excel to undergraduate scientists
There are two files in this repository. The main Excel Escape Room is the tool currently used in the Southampton University Undergraduate Teaching labs, introducing Excel skills as part of a practical chemistry module.
A full study has been published into the effectiveness of this learning tool, which can be accessed open source from https://doi.org/10.1021/acs.jchemed.5c01679
The second file (ViCEPHEC) is a conference-ready Escape Room, which has been designed to teach the user how to build Escape Rooms like this one. The Escape Rooms are all built using native Excel functions. This ViCEPHEC version, presented as part of a workshop for ViCEPHEC2026, contains a gamified approach to teaching the user how to develop their own smart worksheet, providing demonstrations of use and creating the necessary puzzle mechanisms. 
If you are interested in using any of these, or developing your own, feel free to drop me a message, I'm always happy to help!
